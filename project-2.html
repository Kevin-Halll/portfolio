Amber Eats

This page contains the case study of Amber Eats Web App which includes the Project Overview, Tools Used and Links to the Github repository.

Github Link

Project Overview

The objective of the Amber Eats project is to create an online ecommerce restaurant platform for a company to be able to showcase their array of dishes on their menu so that customers can efficiently browse through those dishes by categories and make their purchase. Certain categories of dishes on the menu will also activate an upsale modal for compimentary orders when a customer adds that dish to purchase.

The Amber Eats project was done as a team in which I coled where my main focus was with the design. I was able to gain inpiration from other online restaurants such as Sepia to create the design for this project in Figma.
Feel free to take a look at the Figma design that I created for this project right here.
